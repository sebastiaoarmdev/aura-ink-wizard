# 🖋️ Aura Ink Studio - Chatflow Wizard

> **Link do Projeto:** [Visualizar Projeto Gratuito](https://sebastiaoarmdev.github.io/aura-ink-wizard/)

Este projeto é um simulador interativo de triagem e conversão para o **Aura Ink Studio**, um estúdio de tatuagem premium. Ele guia o usuário por um fluxo de perguntas personalizadas e, ao final, formata os dados e direciona o cliente para o WhatsApp do estúdio com um lead 100% qualificado.

## ✨ Funcionalidades
- **Fluxo Dinâmico:** Respostas integradas que avançam o usuário de forma fluida.
- **Deep Linking:** Geração automática e tratada (`encodeURIComponent`) de mensagens personalizadas para a API do WhatsApp.
- **Interface Premium:** Design minimalista, responsivo e focado na experiência do usuário (UX).

## 🛠️ Tecnologias Cogitadas / Utilizadas
- HTML5 Semântico & CSS3 (ou Tailwind CSS)
- JavaScript / TypeScript (Clean Code & Separação de Responsabilidades)
- Deploy Automatizado via GitHub Pages
